

#  Blin 

**Blin** es una herramienta web ligera y minimalista diseñada para generar fondos con efectos de desenfoque (blur) de forma rápida y sencilla. Ideal para desarrolladores y diseñadores que buscan crear capas visuales modernas o efectos de "Glassmorphism" sin complicaciones.

 **[Ver Demo en Vivo](https://uplays.github.io/Blin/)**

---

##  Características

* **Simplicidad**: Interfaz limpia enfocada en la facilidad de uso.
* **Generación en Tiempo Real**: Ajusta los parámetros y visualiza los cambios instantáneamente.
* **Ligero**: Desarrollado con tecnologías web estándar para una carga ultrarrápida.
* **Open Source**: Código abierto disponible para la comunidad.

##  Tecnologías utilizadas

Este proyecto está construido con:

* **HTML5** - Estructura semántica.
* **CSS3** - Estilos y efectos de desenfoque avanzados.
* **JavaScript (Vanilla)** - Lógica de interacción sin dependencias pesadas.
* **GitHub Pages** - Despliegue y hosting.

##  Instalación y Uso Local

Si deseas ejecutar este proyecto localmente o contribuir al desarrollo, sigue estos pasos:

1. **Clona el repositorio:**
```bash
git clone https://github.com/uplays/Blin.git

```


2. **Navega al directorio:**
```bash
cd Blin

```


3. **Abre el archivo principal:**
Simplemente abre el archivo `index.html` en tu navegador preferido.

##  Cómo usar

1. Abre la aplicación en tu navegador.
2. Utiliza los controles deslizantes o selectores para ajustar la intensidad del desenfoque y los colores de fondo.
3. Copia el código CSS generado para implementarlo en tu propio proyecto.
