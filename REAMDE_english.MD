# Blin

**Blin** is a lightweight and minimalist web tool designed to generate backgrounds with blur effects quickly and easily. Ideal for developers and designers looking to create modern visual layers or "Glassmorphism" effects without complications.

**[View Live Demo](https://uplays.github.io/Blin/)**

---

## Features

* **Simplicity**: Clean interface focused on ease of use.
* **Real-Time Generation**: Adjust parameters and visualize changes instantly.
* **Lightweight**: Developed with standard web technologies for ultra-fast loading.
* **Open Source**: Open source code available to the community.

## Technologies Used

This project is built with:

* **HTML5** - Semantic structure.
* **CSS3** - Advanced styles and blur effects.
* **JavaScript (Vanilla)** - Interaction logic without heavy dependencies.
* **GitHub Pages** - Deployment and hosting.

## Installation and Local Use

If you want to run this project locally or contribute to its development, follow these steps:

1. **Clone the repository:**

```bash
git clone https://github.com/uplays/Blin.git

```

2. **Navigate to the directory:**

```bash
cd Blin

```

3. **Open the main file:**
Simply open the `index.html` file in your preferred browser.

## How to Use

1. Open the application in your browser.
2. Use the sliders or selectors to adjust the blur intensity and background colors.
3. Copy the generated CSS code to implement it in your own project.
